<objective>
Wire up smooth-scroll anchor navigation so visitors can jump between the three primary sections of the single-page site (Allan / Lachlan / FAQ) using the header nav. This plan delivers everything required to satisfy NAV-04 EXCEPT the `#lachlan` target itself — that target is created when Plan 02 inserts the MentorIntro component with `id="lachlan"`.

And without `id="allan"` / `id="faq"` on the existing sections, the new nav links would have nothing to scroll to. This plan flips both switches and rewires the header in one focused pass.
