  <action>
    1. Initialize Astro project in the current directory using the minimal template:
       ```
       pnpm create astro@latest . -- --template minimal --install --no-git
       ```
       Use `.` as the directory since we are already in the project root. Use `--no-git` because git will be initialized separately. If the command requires confirmation to use the current directory, accept it.

    2. Install Tailwind CSS v4 and its Vite plugin:
       ```
       pnpm add tailwindcss @tailwindcss/vite
       ```

    3. Install Inter Variable font:
       ```
       pnpm add @fontsource-variable/inter
       ```

    4. Configure astro.config.mjs -- replace the default content with:
       ```typescript
       // @ts-check
       import { defineConfig } from "astro/config";
       import tailwindcss from "@tailwindcss/vite";

       export default defineConfig({
         vite: {
           plugins: [tailwindcss()],
         },
       });
       ```

       CRITICAL: Do NOT use `@astrojs/tailwind` (deprecated for v4). Do NOT create a `tailwind.config.js` file. Tailwind v4 is CSS-first.

    5. Create a simple SVG favicon at public/favicon.svg -- a simple gold "E" on dark background or a minimal geometric mark.

    6. Verify the scaffolding by checking:
       - `package.json` contains `tailwindcss`, `@tailwindcss/vite`, `@fontsource-variable/inter`
       - `astro.config.mjs` uses the Vite plugin pattern (NOT integrations)
       - No `tailwind.config.js` exists
       - No `@astrojs/tailwind` in package.json
  </action>
  <verify>
    Run `pnpm astro check` (if available) or `pnpm build` to confirm the project compiles.
    Verify package.json contains: tailwindcss, @tailwindcss/vite, @fontsource-variable/inter.
    Verify NO tailwind.config.js exists.
    Verify astro.config.mjs contains `vite: { plugins: [tailwindcss()] }`.
